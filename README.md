# 🍔 Speedee AI – Kitchen Load Simulator

## Overview
This is a prototype system that simulates real-time kitchen operations and recommends crew allocation based on predicted load.

## Features
- Predicts kitchen load using a machine learning model
- Live dashboard (Flask)
- Trend-based alert system (avoids reacting to noise)
- Crew movement recommendations

## How to run

1. Install dependencies:
pip install -r requirements.txt


3. Open in browser:
http://127.0.0.1:5000/


## Notes
This project uses simulated data.  
Next step would be connecting it to real POS and kitchen activity data.
